Step 1 — Push to GitHub
# In your terminal, inside the project folder:
git init
git add .
git commit -m "GetBlock Outbound Intelligence v1"
git remote add origin https://github.com/YOUR_USERNAME/getblock-outbound.git
git push -u origin main
Step 2 — Deploy on Railway
1. Go to railway.app → New Project
2. Click "Deploy from GitHub repo"
3. Select your getblock-outbound repo
4. Railway auto-detects Node.js — no config needed
5. Click Deploy — takes ~90 seconds
6. Go to Settings → Domains → Generate Domain
7. You get a URL like getblock-outbound.up.railway.app
Step 3 — Update your webhook URLs
After Railway gives you a live URL, update these two places:

In Clay HTTP API step:
https://YOUR-APP.up.railway.app/api/webhook/inbound

In Instantly / your email tool reply webhook:
https://YOUR-APP.up.railway.app/api/webhook/reply

That's it — Clay pushes leads in, Instantly pushes replies in, everything lands in this app automatically.
